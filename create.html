Registration Guidelines

Who Can Be Registered

  • Cell Leaders (Akagari)
  • Sector Executive Secretaries (umurenge)
  • District Mayors (Akarere)
  • Province Governors (Intara)
  • System Administrators

Required Information

  • Valid Rwanda National ID (16 digits)
  • Active Rwanda mobile number
  • Full name as in official documents
  • Correct administrative assignment

Verification Process

  • National ID validated against government database
  • Phone number verified via SMS code
  • Administrative hierarchy cross-checked
  • Approval from higher-level supervisor

Security Protocols

  • All accounts start with temporary password
  • Password change required on first login
  • Two-factor authentication enabled
  • All actions logged for audit trail
